# apple-workflows

Manage Apple Notes, Reminders, and Shortcuts with AI agents via a local MCP server on macOS.

## Plugin Reference

@plugins/apple-workflows/README.md

## MCP Setup

Add the MCP server from `gemini-settings-snippet.json` to your Gemini settings.

> **Note:** Hook-based automation requires Claude Code. Hooks are not executed by Gemini CLI.
