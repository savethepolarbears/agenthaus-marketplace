# ActivePieces Flow Pilot

A connector-backed agent plugin pack for **designing, creating, refactoring, publishing, enabling, and operating ActivePieces flows**.

## Included skills

1. **Discover capabilities** — inventories flows, pieces, actions, triggers, and auth
2. **Design blueprint** — turns a natural-language brief into a build plan
3. **Build or refactor flow** — mutates flows safely step by step
4. **Publish, enable, and run** — handles release and trigger-aware execution strategy

## Included specialist agents

- **Flow Architect** — plans the automation, data model, and control flow
- **Flow Operator** — performs low-risk mutations, publishing, enabling, disabling, and note management
- **Flow Auditor** — inspects structure, auth, invalid steps, dead branches, and release readiness
