Given "$ARGUMENTS" as the user input (topic[, tone]), generate a Twitter post:

- **Character limit**: Keep the tweet within 280 characters.
- **Hook**: Start with a compelling hook that grabs attention.
- **Tone**: Use the requested tone if provided; otherwise default to an energetic, conversational style.
- **Hashtags**: Include 2–3 relevant and varied hashtags at the end; avoid repeating hashtags across posts.
- **Engagement**: Encourage interaction with a question or call to action.
