# Content Creation — Brand Voice & Format Library

## Brand Voice Principles

All Black Bear Media content follows these principles:
- **Conversation-first**: Write like you're telling a friend, not broadcasting
- **Authentic local knowledge**: Position as insider tips, not tourist guide regurgitation
- **Opinion-driven hooks**: Take a stance — "Skip this, do this instead"
- **Utility over inspiration**: At least 30% of Greece cluster posts must be utility-led (planning, tips, warnings, itineraries)

## Format Mix Rules (NON-NEGOTIABLE)

1. No brand runs 3+ consecutive scenic-only posts without a utility post breaking the pattern
2. Every flagship week includes at least 1 planning carousel + 1 comparison/utility reel
3. Germany content shifts from scenic singles to route and utility-led carousels
4. At least 30% of Greece cluster content must be utility-led

## Caption SOP by Format

### Scenic Still
- One vivid descriptive line
- One useful/practical line
- One simple CTA (save, comment, or send — pick ONE)

Example:
```
Golden hour paints Oia's caldera like nothing else on earth.

Best viewed from the castle ruins — arrive 45 minutes before sunset to grab a spot.

Save this for your Santorini trip planning.
```

### Utility Carousel
- Lead with the promise/benefit in the first line
- Keep every frame task-oriented
- End with a clear save CTA

Example:
```
3 beaches in Crete that locals keep to themselves.

Most visitors only hit Elafonissi and Balos. These three are just as beautiful — without the crowds.

Save this before your trip.
```

### Comparison Reel
- Open with the contrast in the first second
- End with a vote CTA

### Gear/Creator Reel
- Lead with the user problem, NOT the product name

## Threads-Specific Writing Rules

Threads content is NOT copy-pasted from Facebook. It follows its own rules:

1. **Under 500 characters** — punchy, conversational, opinion-driven
2. **No hashtags** — use topic tags for discoverability instead
3. **No links in post body** — links go in first comment only
4. **Authentic tone** — write like a person, not a brand
5. **Images outperform text-only** — always include an image when possible
6. **Top-performing formats**: curated lists, local storytelling, debate prompts, utility micro-guides
7. **Strong hooks**: Start with a bold statement, question, or hot take

Example Threads post (Athens Secrets):
```
The Acropolis now requires timed entry tickets.

Most tourists don't know this until they're standing in line being turned away.

Book at least 2 days ahead on the official site — the early morning slot (8am) has the smallest crowds and best light for photos.
```

## Channel-Specific Adaptations

| Channel | Job | Key Rule |
|---------|-----|----------|
| **Facebook** | Core distribution + traffic bridge | Strong first lines, clean first-comment article links |
| **Instagram** | Brand equity + saves + discovery | Push utility carousels; scenic-only posting is insufficient |
| **Threads** | Companion awareness | Rewrite copy; never paste Facebook captions |
| **LinkedIn** | Founder/business authority | Only authority, planning, business-relevant content |
| **Pinterest** | Evergreen discovery | Direct article links, searchable titles, not generic copy |
| **Google Business** | Local-intent support | Write like a local recommendation, not a social caption |

## Seasonal Content Hooks (Greece)

- **March–April**: Greek Orthodox Easter, shoulder season tips, "beat the crowds" angles
- **May–June**: Peak season prep, beach guides, ferry schedules, heat warnings
- **July–August**: Peak season survival, crowd avoidance, late-night dining, water sports
- **September–October**: Shoulder season sweet spot, harvest festivals, wine season, hiking
- **November–February**: Off-season travel, local life, holiday traditions, budget travel

## Content Research Workflow

1. **Check Google Drive first** for existing content, scheduled posts, and brand assets to avoid duplication
2. **Search web** for current seasonal hooks, local events, trending travel topics
3. **Check VistaSocial** via `searchPosts` to see what's been published recently — avoid repeating topics within 2 weeks
4. **Draft content** following the format library and caption SOPs above
5. **Adapt per channel** — never copy-paste across networks
