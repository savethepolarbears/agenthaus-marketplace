# Marketplace Rules

Rules for the AgentHaus marketplace registry.

## Plugin Registration

1. Validate plugin structure before registering
2. Use only standard categories: `devops`, `productivity`, `content`, `qa`, `docs`, `cloud`, `database`, `rag`, `knowledge`, `utility`
3. Descriptions: Keep under 150 characters, start with action verb
4. Source paths: Use relative paths from `marketplace.json`

## Categories

| Category       | Use For                                    |
| -------------- | ------------------------------------------ |
| `devops`       | CI/CD, deployment, infrastructure          |
| `productivity` | Task management, scheduling, notifications |
| `content`      | Content creation, social media, writing    |
| `qa`           | Testing, quality assurance, validation     |
| `docs`         | Documentation, knowledge retrieval         |
| `cloud`        | Cloud platform integrations                |
| `database`     | Database operations, migrations            |
| `rag`          | RAG patterns, knowledge synthesis          |
| `knowledge`    | Note-taking, wikis, knowledge bases        |
| `utility`      | General-purpose tools                      |
