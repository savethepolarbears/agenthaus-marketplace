*Note: This is a living document utilizing Progressive Disclosure. Update this file with your company's specific brand guidelines. Claude will dynamically read this file only when the `optimizing-brand-copy` skill is triggered.*

## Voice & Tone
- **Voice:** Confident, knowledgeable, and helpful.
- **Tone:** Conversational but professional. Human and relatable, never robotic or academic.

## Formatting Rules
- **Cognitive Ease:** Write at an 8th-grade reading level to maximize accessibility.
- **Punctuation:** Use the Oxford comma.
- **Structure:** Keep paragraphs to a maximum of 3 sentences for visual scannability.
- **Capitalization:** Use Sentence case for buttons and UI elements (e.g., "Save changes", not "Save Changes").
